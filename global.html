providerCallback(err, res)

This callback will be called after executing this method. The first parameter will contain an error object if an error occurred during execution while the second parameter will contain the result from the execution if it was successful.

Parameters:
Name Type Description
err Object

the error object upon unsuccessiful execution.
res Object

the result object upon successiful execution.
